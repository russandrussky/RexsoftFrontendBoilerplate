Секции с сайдбарами

У Вас есть возможность выбора различных вариантов. Вы можете использовать оба сайдбара, или только один. Также Вы можете разместить их оба сдева или оба справа. Есть возможность размещения сайдбаров в коде страницы как до контента, так и после. Размещение по умолчанию — после контента. Разместив сайдбар до контента добавьте класс .before__content.
